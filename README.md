# Tul-X

Tul-X is a lightweight interpreter-based programming language built in C, designed to demonstrate core concepts of compiler and interpreter design such as lexical analysis, parsing, AST generation, and expression evaluation.

---

## Features

- Lexical Analysis (Tokenization)
- Recursive Descent Parser
- Abstract Syntax Tree (AST) Generation
- Expression Evaluation Engine
- Operator Precedence & Associativity Handling
- Modular and Clean Architecture
- Basic Error Handling

---

## Tech Stack

- Language: C
- Concepts: Compiler Design, Interpreter, Parsing, AST

---

## How It Works

1. Source code is converted into tokens using the scanner  
2. Tokens are parsed into an Abstract Syntax Tree (AST)  
3. The interpreter evaluates the AST  
4. Final result is printed as output
